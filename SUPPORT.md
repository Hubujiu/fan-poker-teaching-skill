# Support

## Questions and usage help

Open a GitHub Discussion when Discussions are enabled, or create an issue with a minimal description of:

- the Agent or CLI you use
- the prompt you gave it
- the generated HTML or relevant `cardData`
- the browser and device where the problem appears

## Bugs

Use the bug report template for animation clipping, blank cards, broken input handling, syntax errors, or generated files that no longer work as standalone HTML.

## Feature ideas

Use the feature request template and include a concrete teaching use case. Features that remain reusable across subjects are more likely to fit the base skill.

## Security

Do not post exploit details publicly. Follow [SECURITY.md](./SECURITY.md).
